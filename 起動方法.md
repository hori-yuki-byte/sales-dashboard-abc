# 商談ログ分析ダッシュボード 起動方法

## 初回セットアップ（1回だけ）

ターミナルを開いて以下を実行：

```bash
cd /Users/horitan/Desktop/AI用/sales-dashboard
pip install -r requirements.txt
```

## 起動

```bash
cd /Users/horitan/Desktop/AI用/sales-dashboard
streamlit run app.py
```

ブラウザが自動で開きます（http://localhost:8501）

## 使い方

1. GoogleスプレッドシートをCSVでダウンロード
   - ファイル → ダウンロード → カンマ区切り形式（.csv）

2. サイドバーからCSVをアップロード

3. 集計期間を選ぶ（3日 / 1週間 / 2週間 / 全体）

4. AI分析を使う場合はClaude API Keyを入力
   - https://console.anthropic.com でAPIキーを取得

## Claude API Keyの取得

1. https://console.anthropic.com にアクセス
2. サインアップ / ログイン
3. 「API Keys」→「Create Key」
4. 発行されたキーをダッシュボードに入力
